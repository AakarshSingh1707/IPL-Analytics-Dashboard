# IPL Analytics Dashboard (Power BI)

## 📊 Project Overview
This project analyzes IPL data using Power BI to generate insights on team performance, player statistics, and venue trends.

## 🚀 Features
- Total matches and runs analysis
- Team-wise performance
- Venue analysis
- Top batsmen and bowlers

## 🛠 Tech Stack
- Power BI
- CSV datasets
